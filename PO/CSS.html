Cascading Style Sheets, CSS dus. Hierin betekend Style "opmaak", dat zegt dus al heel veel over wat het doet. Op het plaatje onder op de pagina zie je hoe het in zijn werk gaat. Voor je de { } opent maak je duidelijk waar je iets wilt aanpassen, welke tekst, plaatje of welk blok. Vervolgens zet je in de { } wat je wilt doen, dit kan bijvoorbeeld een kleurverandering zijn, maar het kan ook duidelijk maken waar het zou moeten staan, of hoe groot.

Waar ikzelf CSS nu voor gebruik, is het maken van deze website bijvoorbeeld. Onder andere om aan te geven dat deze tekstgroen is of dat deze tekst een ander lettertype heeft. Er zijn heel veel opties met CSS, veel uitgebreider dan wat ik liet zien met wat tekst. Wil je er zelf meer over weten? Bekijk dan deze videoof ga zelf op zoek op w3schools.
