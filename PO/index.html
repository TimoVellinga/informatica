HTML

HyperText Markup Language, HTML. Dit is de standaard opmaaktaal voor het wereldwijde web. Deze code word gebruikt om een structuur te geven aan een website. Hiermee kan je bijvoorbeeld opsommingspunten of tabellen maken, illustraties weergeven of een tekst typen. Wil je meer leren over HTML? Ga dan naar w3schools.

CSS, de make-up doos van HTML. De officiele naam is Cascading Style Sheets. Hiermee kun je bepalen wat je wilt doen met de inhoud die je in HTML hebt gezet. Als je ervoor zorgt dat CSS met HMTL kan praten, kan je bijna alles doen met je inhoud. Je kan bijvoorbeeld bepalen welke kleur het moet zijn, hoe groot, welk letterype en nog veel meer. Meer informatie op w3schools.
