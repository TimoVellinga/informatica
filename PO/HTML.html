HyperText Markup Language, HTML dus. Elke website die je ziet heeft wel iets te maken met HTML, het is onmisbaar bij het maken van een website. Hiermee kan je precies zeggen tegen de site wat je wilt, waar, hoe en nog veel meer. Als ik wil dat dit kleine stukje tekst schuin staat, dan geef ik dat aan met < i > en dan doet hij dat. Er zijn nog veel meer opties zoals dit, ontdek deze op w3schools.

Vaak word "coderen" gezien als iets heel ingewikkelds, maar als je jezelf erin verdiept is het vrij simpel. Je kan het altijd uitproberen en gaandeweg leer je al meer. Het belangrijkste vind ikzelf opletten, als je een fout maakt kan het lang duren voordat je hem oplost. In HTML zet je vaak de inhoud van je pagina, de blokken, text enzovoort. Vervolgens vertel je in CSS hoe het moet staan en hoe het eruit zou moeten zien. Wil je meer weten? Bekijk w3schools.
